# Analog_Pulse_Duration_Detector
Overview

This project implements an analog pulse duration detection circuit that turns ON an LED only when an input pulse remains HIGH for a minimum threshold time. Shorter pulses are ignored to prevent false triggering.

Technologies / Tools

LTspice

Operational Amplifiers

RC Timing Networks

Diodes and Transistors

Key Features

Detects minimum pulse width

Ignores short noise pulses

Automatic reset when input pulse goes LOW
